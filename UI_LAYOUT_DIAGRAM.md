## Design System Colors

### Background Colors:
- `bg-black` - Main background
- `bg-white/5` - Subtle card background
- `bg-white/10` - Hover states

### Text Colors:
- `text-white` - Primary text (100%)
- `text-white/90` - Secondary text
- `text-white/70` - Tertiary text
- `text-white/60` - Muted text
- `text-white/40` - Very muted text (labels, meta)

### Border Colors:
- `border-white/20` - Strong borders (inputs, buttons)
- `border-white/15` - Standard borders (panels, cards)
- `border-white/10` - Subtle borders

### Accent Colors:
- `text-green-500` - Success/completed (âœ“)
- `text-blue-400` - Active/running
- `text-red-400` - Error
- `text-yellow-400` - Warning

### Interactive States:
- `hover:bg-white` - Button hover (inverts to white bg, black text)
- `hover:bg-white/5` - Card hover
- `hover:text-white` - Link hover
- `disabled:opacity-40` - Disabled state

## Animation & Transitions

- Panel transitions: None (fixed grid)
- Node expansion: Smooth collapse/expand
- Loading states: Subtle opacity changes
- Hover effects: Quick (150ms) color transitions
- Status updates: Instant (no animation needed for real-time data)

## Accessibility Notes

- High contrast (white on black)
- Clear hierarchy with size/weight/opacity
- Interactive elements have clear focus states
- Expandable sections use semantic HTML
- Keyboard navigation supported (Enter to send, Tab to navigate)
